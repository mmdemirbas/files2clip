# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please report it by opening a [GitHub issue](https://github.com/mmdemirbas/files2clip/issues) or emailing the maintainer directly.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Scope

files2clip is a local CLI tool that reads files and writes to the system clipboard. It does not make network connections, run servers, or process untrusted input from remote sources.
